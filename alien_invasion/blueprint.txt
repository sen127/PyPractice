alien_invasion game

Functionality
User Ship
	A user character ship always starts from the middle and able to move left and right
	Player ship stays in the same 1d and not to be able to move in 2d
	Player ship be able to fire continuously when pressed space
	Speed of the bullets can be changed temporary when ship catches a upgrade module represented by a star

Alien Ships
	Alien ships keep coming from top - opposite direction and able to move in 2D (down and left/right)
	Each alien ship has a health/power count to track hits before destruction
	Descending speed increases with each level
	Alien ships fire bullets at player ship

Power-ups
	Star modules temporarily boost bullet speed
	Shield modules reduce incoming damage

General
	Level counter affects alien ship speed and spawn rate
	Score tracking for player progression
	Collision detection between bullets, ships, and power-ups
